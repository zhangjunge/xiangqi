# 象棋小课堂 - 规格说明

## 目标人群
- 7岁左右
- 已完全理解中国象棋规则
- 需要通过小游戏巩固常见走法和吃子技巧

## 产品目标
- 用短小关卡的方式练习走法与吃子
- 提供清晰、正向的即时反馈
- 支持鼠标与触屏的拖动/点击操作

## 核心功能
- 关卡制教学（循序渐进）
- 目标式通关（多为单步吃子）
- 合法走子校验（车/马/相/仕/帅/炮/兵）
- 拖动走子与点击走子并存
- 关卡提示与“再给我一个提示”按钮
- 关卡重置与坐标显示开关
- 过关动画与基础音效反馈

## 交互说明
- 选择棋子：点击红方棋子
- 移动方式：
  - 点击棋子后点击目标格子
  - 拖动棋子到目标格子后释放
- 错误反馈：显示原因（如“车不能隔子”）
- 过关反馈：显示“完成目标”提示

## 关卡列表（15关）
1. 直线吃子（车）
2. 炮隔子打
3. 马走日
4. 相走田
5. 仕守九宫

7. 过河兵

9. 马腿检查
10. 炮的对齐
11. 仕走对角
12. 车直线清路
13. 挡炮练习（两步目标）
14. 牵制小练习
15. 双将军入门

## 技术实现
- 静态网页：`index.html` + `style.css` + `script.js`
- 适配 GitHub Pages
- 无后端依赖
- 使用 Web Audio 生成基础提示音
